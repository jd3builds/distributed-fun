## Setup Instructions

1. Clone the repository:

    ```bash
    git clone https://github.com/yourusername/my_project.git
    cd my_project
    ```

2. Run the setup script:

    ```bash
    ./setup.sh
    ```

3. Run code!:

    ```bash
    python src/example.py
    ```
